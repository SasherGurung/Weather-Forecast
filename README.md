Weather Forecast App

Developed a modern, responsive Weather Forecast web applicatio built with Next.js, Tailwind CSS and Axios. It provides real-time weather updates by integrating the OpenWeatherMap API, offering users detailed information such as temperature, humidity, wind speed, cloudiness, sunrise, and sunset times.

Features:
* Fetches live weather data for any city worldwide using the OpenWeatherMap API.
* Displays comprehensive weather details, including:
    * Current temperature & feels-like temperature
    * Humidity and atmospheric pressure
    * Wind speed and cloudiness
    * Sunrise and sunset times
* Dynamic weather icons that automatically update based on current weather conditions.
* Fully responsive and visually appealing interface designed with Tailwind CSS.
* Input validation and error handling for incorrect city names.
* Built with Next.js for server-side rendering and performance optimization.
* Uses Axios for efficient and reliable API requests.

Tech Stack:
* Next.js – React framework for building fast, SEO-friendly Eeb apps.
* Tailwind CSS – CSS framework for clean styling.
* Axios – Promise-based HTTP client for handling API requests.
* OpenWeatherMap API – Provides accurate weather data worldwide.
